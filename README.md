# Digital Flavour Style Guide

Front-end style guide for the Digimark marketing site, built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui components.

## Local development

1) Install dependencies
```sh
npm install
```

2) Run the dev server
```sh
npm run dev
```

3) Lint (optional)
```sh
npm run lint
```

## Project structure
- `src/` React app code (pages, components, hooks, styles)
- `public/` static assets
- `tailwind.config.ts` Tailwind and design tokens
- `vite.config.ts` Vite build/dev configuration
